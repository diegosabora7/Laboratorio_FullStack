================================================================================
CONSEJOS GENERALES DE BÚSQUEDA
================================================================================

1. PRIORIZAR FUENTES:
   - Documentación oficial (Spring, React, TanStack) → siempre la más confiable
   - Baeldung (para Java/Spring) → tutoriales completos y actualizados
   - Stack Overflow → soluciones probadas por la comunidad
   - GitHub Issues → para bugs específicos de librerías

2. FILTRAR POR VERSIÓN:
   - Agregar "Spring Boot 3" o "React 18" a las búsquedas
   - Filtrar resultados por fecha (último año) para evitar soluciones obsoletas

3. PALABRAS CLAVE EFECTIVAS:
   - Usar el nombre exacto de la excepción (ej: "ObjectOptimisticLockingFailureException")
   - Usar el nombre de la anotación (ej: "@CacheEvict", "@CircuitBreaker")
   - Incluir "example" o "tutorial" para resultados prácticos

4. IDIOMA:
   - Buscar siempre en INGLÉS para mayor cantidad de resultados
   - La documentación oficial está en inglés

5. STACK OVERFLOW:
   - Buscar con [tag] para filtrar: [spring-boot] [jpa] N+1
   - Ordenar por votos para ver las respuestas más validadas
   - Leer los comentarios — a veces la mejor solución está ahí
